Servidor HTTP Simple en Python

Este servidor está ejecutándose utilizando el módulo http.server de Python, una herramienta simple pero potente para desplegar servidores web básicos.

Características del Servidor Python

  • Implementación simple usando bibliotecas estándar
  • Ideal para desarrollo y pruebas
  • Fácil de configurar y personalizar
  • Perfecto para proyectos educativos

Ejemplo de código

from http.server import HTTPServer, SimpleHTTPRequestHandler

def run(server_class=HTTPServer, handler_class=SimpleHTTPRequestHandler):
    server_address = ('', 8000)
    httpd = server_class(server_address, handler_class)
    print("Servidor iniciado en puerto 8000...")
    httpd.serve_forever()

if __name__ == "__main__":
    run()
